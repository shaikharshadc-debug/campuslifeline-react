# CampusLifeLine (React + Vite + Tailwind)

This is a ready-to-run frontend prototype for the CampusLifeLine donor network.  
Run locally:

```bash
npm install
npm run dev
```

Then open http://localhost:5173 (or the URL printed by Vite).

This project uses Vite + React + Tailwind. The main app is in `src/App.jsx`.
